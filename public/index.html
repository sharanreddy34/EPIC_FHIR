ATLAS Engine JWKS Endpoint

This page hosts the JSON Web Key Set (JWKS) for ATLAS Engine's Epic FHIR API integration.

JWKS Endpoint URL

The JWKS endpoint is available at:

https://gsiegel14.github.io/ATLAS-EPIC/.well-known/jwks.json

How to Use

In Epic's developer portal:

  1. Use this URL for both Production and Non-Production JWK Set URL fields
  2. Make sure "Backend Systems" is selected as the user type
  3. Check "Uses OAuth 2.0"

JWT Configuration

When generating JWT tokens, include the JKU header pointing to this URL:

python generate_jwt.py --jwks-url "https://gsiegel14.github.io/ATLAS-EPIC/.well-known/jwks.json"
